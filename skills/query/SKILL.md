# /tencent-coding-plan-hud:query — 用量查询

查询并显示腾讯云 CodingPlan 的当前用量数据。

## 使用方式

Arguments: `$ARGUMENTS`

### 无参数 — 标准查询

直接运行 `/tencent-coding-plan-hud:query` 显示完整用量信息：

1. 获取当前 HUD 配置
2. 使用 MCP 工具 `mcp__tencent-coding-plan-hud__get_usage_detail` 查询
3. 显示详细用量报告，包括：
   - 套餐信息（名称、类型、价格）
   - 剩余天数和有效期
   - 5小时周期用量
   - 每周用量
   - 每月用量

### 带参数 — HUD 显示

如果需要只显示 HUD 格式：
```
/tencent-coding-plan-hud:query hud
```

## 输出格式

### 详细格式

```

💰 价格: ¥200/月
📊 状态: Normal
⏰ 剩余天数: 10 天
📅 有效期: 2026-03-25 09:59:58 ~ 2026-04-25 09:59:58
============================================================

🔄 5小时周期:
   时间范围: 2026-04-15 06:13:50 ~ 2026-04-15 11:13:50
   用量: 90 / 6,000 tokens
   使用率: 1.50%
   剩余: 5,910 tokens

📅 每周:
   用量: 2,124 / 45,000 tokens
   使用率: 4.72%
   剩余: 42,876 tokens

📆 每月:
   用量: 19,492 / 90,000 tokens
   使用率: 21.66%
   剩余: 70,508 tokens
============================================================
```

### HUD 格式

根据配置的 UI 布局显示简洁的用量信息。

## 错误处理

如果 API 未配置或调用失败，显示：
```
API未配置或者已失效过期，请输入/tencent-coding-plan-hud:setup配置。
```

## 数据刷新

- 数据缓存在内存中
- 根据配置的刷新频率自动更新
- 手动查询会触发立即刷新（如果缓存过期）
