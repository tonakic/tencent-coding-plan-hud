# /tencent-coding-plan-hud:configure — 显示设置配置

配置 HUD 的 UI 布局和数据刷新频率。

## 配置选项

Arguments: `$ARGUMENTS`

### A. UI 布局选择（单选）

| 选项 | 布局类型 | 描述 |
|------|---------|------|
| `expanded-simple` | 展开（简洁） | 单行显示，简洁格式（默认） |
| `expanded-detailed` | 展开（详细） | 单行显示，包含具体数值 |
| `stacked-simple` | 堆叠（简洁） | 多行显示，简洁格式 |
| `stacked-detailed` | 堆叠（详细） | 多行显示，包含具体数值 |

**布局示例：**

展开（简洁）：
```
上下文用量█████░░░░░ 45%|5h用量██░░░░░░░░ 25% |周用量██░░░░░░░░ 25% |月用量██░░░░░░░░ 25% |套餐: Coding Plan Pro |刷新时间：2026-04-15 12:30:39
```

堆叠（详细）：
```
上下文用量（90K/200K）█████░░░░░ 45%
5h用量(1500/6000)██░░░░░░░░ 25%
周用量(11250/45000)██░░░░░░░░ 25%
月用量（22500/90000）██░░░░░░░░ 25%
套餐: Coding Plan Pro
刷新时间：2026-04-15 12:30:39
```

### B. 刷新频率选择（单选）

| 选项 | 频率 |
|------|------|
| `0.5` | 30秒/次 |
| `1` | 1分钟/次 |
| `2` | 2分钟/次（默认） |
| `3` | 3分钟/次 |
| `5` | 5分钟/次 |
| `10` | 10分钟/次 |

## 配置方式

### 命令行方式

直接指定配置：
```
/tencent-coding-plan-hud:configure layout=stacked-detailed refresh=5
```

### 交互方式

如果没有参数，显示选项让用户选择：

1. 显示 UI 布局选项（A）
2. 显示刷新频率选项（B）
3. 使用 MCP 工具 `mcp__tencent-coding-plan-hud__configure_hud` 应用配置
4. 显示新的 HUD 示例

## 进度条颜色规则

- **绿色**：使用率 0-69%
- **橙色**：使用率 70-89%
- **红色**：使用率 90-100%

## 注意事项

- 需要先完成 setup 配置 API 密钥
- 配置更改即时生效
- 配置保存在 `~/.claude/tencent-coding-plan-hud/config.json`
